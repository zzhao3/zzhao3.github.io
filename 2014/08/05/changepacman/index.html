摘要

本文讲述部分Pacman主题优化方法。部分内容来自于网络。详见参考文献部分。
本文不会详细介绍Pacman主题，只是给技术小白们提供一部分主题优化的方法与思路。同时一些比较浅显的方法也不在这里介绍。

补充：后改为jacman主题，所有方法在jacman主题下测试成功。jacman主题详情请见 如何使用jacman主题

更换标题背景

Pacman主题默认的标题背景是橙黄色的，非常好看，但为了追求个性化，我将其换成自己的图片。主要参考了[1]中的方法。

首先给自己的图片放到hexo\themes\pacman\source\img
中，我用了png格式的图片，1273X199，图片名称pb.png。未测试其他格式或大小，欢迎补充。

修改hexo\themes\pacman\_config.yml。在Image底下添加语句

background_img: img/pb.png

保存之后，打开文件hexo\themes\pacman\source\css\_base\variable.styl在//image 下面添加

background-img = hexo-config("background_img")

保存之后，打开文件hexo\themes\pacman\source\css\_partial\header.styl在第五行（可能具体行数有异）找到background，将次行改为

background url(root+background-img)

重新生成（hexo g）之后即可显示为自定义图片。

补充注意：这个方法在2014年亲测好用，但是在2015年6月重新测试的时候失效。经修改，在最后一步（文件header.styl）时，修改background 为

background: url("/img/noise.png")

之后成功。测试其他文件时，发现似乎类似的导入图片方法都失效。原因未知。

再补充:
使用jacman主题时，该方法适用。

置顶贴

由于hexo中博客的排列是按照博客里面的那个日期来的（其实就是生成博客的日期），因此想要置顶的话，只要给置顶贴的日期改到世界末日那天（比如9999年）即可。

date: 9999-11-03 01:09:10

自定义主页配色

有的朋友很在乎字体、标记等的颜色，希望能够实现自我配置。颜色配置清单在刚才提到的那个文件hexo\themes\pacman\source\css\_base\variable.styl中。这里列出部分颜色的作用。

//Color
color-background = #dddddd//大背景颜色
color-font = #817c7c//博客文字颜色
color-white = #ffffff
color-blue = #2ca6cb//标题颜色
color-theme = #111111//#ea6753主题颜色，鼠标悬停后高亮色
color-font-nav =#ffffff//#E9CD4C鼠标悬停后字体颜色
color-section = #fafafa//博客背景色
color-footer = #1f1f1f//最下部颜色
color-gray = #2E9AFE//#CCC//鼠标悬停前，的颜色
color-heading = #333333//博客正文中的“标题”颜色
color-code = #eee//文章目录背景色

这里颜色是用HTML颜色代码来表示的，从000000（黑色）到ffffff（白色）来表示。具体如何计算这个值，以及寻找自己喜欢颜色的代码，可以在这个网站上来找到。

高级一些的朋友，可以自己定义颜色，然后在hexo\themes\pacman\source\css\ _partial这个文件夹里的文件中，修改页面细节。顺便说一下，这个文件夹里的文件就是描绘页面的文件，更多的自定义配置都可以在这里面找到。
